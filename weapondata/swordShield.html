한손검

무기 소개

경쾌한 이동과 빈틈없는 공격은 물론, 무기를 꺼낸 상태(발도 중)에서도 아이템을 사용할 수 있는 특징을 지닌 무기입니다.

무기 강의crazy (꾸왕)

  1. 장단점
    • 장점
      • 발도 중 슬링어, 아이템 사용가능 : 현 시점 한손검만의 유일한 기능으로 위기대처나 몬스터들의 패턴봉인에 용이하며, 서폿용 무기로 사용되는 이유 중 하나입니다.
      • 백 스텝을 이용한 회피 : 백 스텝에 붙어있는 널널한 회피시간으로 생존에 용이합니다.
      • 자체 단차 액션 : 백 스텝 이후 모아베기가 성공하면 뛰어올라 베기를 자동으로 시전하는데 이때 점프 베기를 이용하면 단차치 쌓기와 자체 단차 액션하기에 용이합니다.
    • 단점
      • 짧은 리치 : 한손검 최악의 단점으로 이 단점 하나 때문에 조금 만 크기가 커도 약점을 맞추기가 힘들고

        누워있는 몬스터한테 모아베기 -> 뛰어올라 베기 -> 폴 배시 콤보가 안 맞는 경우가 자주 있습니다.
      • 가드 베기 : 가드 -> 백, 스탭 -> 모아베기를 사용하는데 있어서 최악의 기술입니다.

        가드 -> 모아베기 시 조금만 실수해도 가드하고 뒤돌아서 후딜이 긴 가드베기를 시전해버립니다.

    • 추천 스킬
      • 장인 : 흰 예리도를 위해 챙겨줍니다.
      • 달인의 재주 : 예리도 소모를 방지하기 위해 챙겨줍니다.
      • 숫돌 사용 고속화, 칼날 연마 : 주로 속성 한손검 할때 예리도의 빠른 회복 및 유지를 위해 챙겨줍니다.
      • 완전 충전 : 커스텀강화는 대부분 회복을 하기 때문에 챙겨주면 딜 상승에 큰 도움이 됩니다.
      • 회피 성능 : 구르기는 물론 백 스텝에도 회피 성능의 효과가 적용됩니다.
      • 심안/탄도 강화 : 폴 배시 운영시 주로 쓰는 무기가 볼보클럽(파예리)라서 심안으로 튕김을 방지해줍니다.
    • 선택 스킬
      • 귀마개
      • 체력증강
      • 가드 성능, 가드 강화

  4. 운영법 및 팁
    • 공통
      • 백 스텝 회피를 이용하자!

        백 스텝 커맨드 파생이 많은 이유는 회피에 있습니다.

        돌진 베기 -> 백 스텝, 구르기 -> 백 스텝, 가드 -> 백 스텝이 가능하기 때문에 회피하기 한층 수월하고

        구르기 -> 백 스텝은 역방향 백 스텝이 가능합니다.

        가드 -> 백 스텝은 백 스텝 타이밍이 늦더라도 가드로 막을 수 있기 때문에 생존에 좋습니다.

        백 스텝으로 회피 후 모아베기 콤보도 가능하기 때문에 적극적으로 이용해주도록 합시다.
        역방향 백 스텝
      • 돌진 베기에는 짧지만 슈퍼 아머가 있다!

        돌진 베기에는 아주 짧지만 슈퍼 아머가 있어서 돌진 베기 -> 백 스텝으로 아주 절륜한 회피가 가능합니다.
      • 올라타기에 성공했다면 3연 공격은 천천히!

        올라타기에 성공 후 단차 마무리 공격 때 천천히 공격키를 입력하시면 바로 눕히지 않고 계속 가격할 수 있습니다.

        이를 이용해서 기절치도 쌓고 딜 누적도 시킬 수 있습니다!
      • 납도와 발도가 빠르다!

        납도와 발도가 빠르기 때문에 구르기 회피가 힘들거 같으면 그냥 납도하고 뛰거나 긴급 회피 해버리자!
    • 모아베기(or 폴 배시)
      • 공격력이 높은 무기를 들자!

        폴 배시 운영은 타수가 적고 속성 데미지 영향을 덜 받기 때문에 공격력이 높은 무기를 들고 가는게 좋습니다.(ex:볼보클럽, 페이탈바이트 등)
      • 거리 조절이 중요!

        한손검은 리치가 짧은데다가 폴 배시의 타점이 좋은 편이 아니기에 정확한 조준이 필요합니다.

        딜 목적으로 백 스텝을 하신다면 막무가내로 백 스텝을 하기보단 모아베기를 할 백 스텝 위치를 신경 써주시면 됩니다.

        만약 백 스텝을 했는데 거리가 멀다 싶으면 모아베기가 아닌 돌진베기로 자리를 다시 잡고 모아베기 각을 잡는게 좋습니다.

        모아베기 시 우클릭을 꾹 누르는 시간에 따라 차징시간이 달라집니다.

        이를 이용해서 소경직으로 잠깐 뒤로 빠진 몬스터를 기다렸다가 공격하는 방법 등으로 사용이 가능합니다.
      • 기타 팁!

        (1) 모아베기 중에 방향키로 미세한 방향조절이 가능합니다.

        (2) 폴 배시 이후 백 스텝이 가능합니다.

        (3) 폴 배시 입력 도중 방향키로 캐릭터가 떨어질 위치 조절이 가능합니다. 이때 만약 폴 배시 -> 백 스텝을 사용하고자 하신다면 화면도 같이 옮겨주시거나 원하는 방향의 역방향키로 백 스텝이 가능합니다.

        (4) 피신복장을 입었다면 뛰어올라 베기 도중에 피격을 당해도 피신 모션이 나오지 않습니다. 즉 폴 배시나 점프 베기를 사용 할 수 있습니다.
    • 사과깍기(선회베기)
      • 공격력보단 속성치가 높은 무기를 들고오자!

        빠른 공격속도로 참격을 무한하게 넣기 때문에 공격력보단 해당 몬스터의 약점속성 무기를 들고 오시는게 좋습니다.
      • 선회베기 -> 찌르기 만 사용하지말자!

        선회베기 -> 찌르기 사이사이에 좌클릭이나 우클릭,회전베기 등을 섞어가면서 써줘야 딜이 더 잘 나옵니다.

        이때 검방패콤보가 나가지 않도록 조심하셔야 합니다.

        선딜도 긴데다 방패공격엔 속성치가 들어가지 않습니다.
      • 방패공격은 자제!

        기절치를 쌓는게 아니라면 방패공격은 최대한 지양하시는게 좋습니다!
      • 잽 잽 원투 잽

        모아베기 한손검이 스트레이트라면 사과깎기 한손검은 잽입니다.
